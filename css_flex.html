flex_레이아웃 속성

flex 레이아웃은 부모자에게 flex를 선언하고 자식요소를 유연하게 배치하는 속성

ul>li 의 경우 ul 에게 flex를 선언하면 li가 가로 정렬됨
이때 li는 item의 속성을 사용한다.

flex_container 속성
속성명 속성값과 의미
display flex - 자손자를 flex로 정렬하겠다 라는 선언문
justify-content 수평정렬에 관한 설정
  • flex-start : 시작점에서부터 정렬(기본값)
  • flex-end : 끝점에서 정렬
  • center : 아이템을 부모자의 가운데 정렬
  • space-between : 처음 아이템은 시작점에 붙고
    마지막 아이템은 끝점에 붙인 상태에서 나머지 아이템을 사이정렬
  • space-around : 모든 아이템들의 좌,우 여백값 통일
aline-item 아이템들의 상,하 여백정렬
(부모자의 높이가 지정되어 있고 아이템들은 부모자 높이값보다 양이 적을때 사용)
  • strech - 아이템들의 위아래 맞춤(기본값)
  • flex-start : 아이템들을 위로 붙임(기존의 박스구조 / 늘어나지 않음)
  • flex-end : 아이템들을 아래로 붙임
  • center : 아이템을 부모 기준으로 가운데 정렬
  • space-beteen : 처음 아이템은 시작점에 붙고
    마지막 아이템은 끝점에 붙인 상태에서 나머지 아이템을 사이정렬 (상,하기준)
  • space-around : 모든 아이템들의 상,하 여백값 통일
flex-direction column(위에서 아래로 수직 정렬)
row (왼쪽에서 오른쪽으로 수평 정렬)
reverse 를 활용하여 역방향 가능 (column-reverse, column-row)
flex-wrap 기본적으로 flex는 자손자를 한줄에만 표시하려 하므로 자손자의 크기는 무시하고 공간에 가둔다 그래서 줄바꿈을 이루어 표기하려면 wrap 으로 묶어준다. flex-wrap : wrap
wrap / no-wrap(기본값) / wrap-reverse
flex-item 속성(자손자에 표기)
order item의 순서를 설정한다
음수까지 허용하며 숫자가 높을수록 뒤로밀림. / 기본값 : 0
flex- grow 증가너비 비율
flex-shink 감소 너비 비율 숫자가 크면 더 많은 너비가 감소한다.
flex-basis item배분 너비 기본값(auto), width값으로 잘 설정하지 않음
flex grow, shink, basis 순서로 기본값 0 auto
aline-self item 에게 각자 지정이 가능하며 aline-item의 세분화 작업
aline-item 의 값을 재 세팅 가능.
