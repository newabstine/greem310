CSS레이아웃

css를 활용하여 공간을 나누듯 배치를 하기위해 여러 속성을 사용한다.

가시속성

display : 태그가 화면에 보이는 방식을 지정한다.

  • none - 화면에서 안보이며 공간도 함께 사라진다 또한 웹접근성으로도 확인할 수 없다.
  • lnline - 특별한 공간의 값을 가지고 있지 않다. width, height 등의 값을 지정할 수 없다. margin, padding 의 경우
    요소를 기준으로 하기 때문에 좌우는 적용이 되나, 상하는 적용되지않는다.
  • lnline-block - inline 과 block 의 중간역할 두 속성의 모든 성질을 가지고 있다.
  • block - 가로값 화면기준 100% 적용된다. 자동 줄바꿈 기능이 있다.

float 속성

자동 줄바꿈의 성질이 있는 블럭을 한줄에 두개 이상의 요소를 배치하고자 할때 사용 / 사전적 의미로 물이나 공중에 '뜨다' 라는 의미로 블럭 요소를 강제로 띄워서 좌우로 배치.

  • left : 왼쪽에 요소를 띄워서 나열한다.
  • right : 오른쪽에 띄워서 나열한다.
  • none : 요소를 띄우지 않음

flex 속성

태그를 수평, 수직으로 자유롭고 유연하게 변동할 수 있는 기능

  • container - 부모요소를 부르는 명칭으로 사용
  • items - 자식요소를 부르는 명칭

container 속성에는 display, flex-flow(묶음기능), flex-derection(수평,수직) justify-content(아이템 수평 정렬), aline-items(아이템 수직 정렬)

items 속성에는 flex, grow(증가너비), shrink(감소너비), basis(크기배분), aline-self(각자조정)

자손자에게만 오롯이 적용된다 후손자 적용안됨

position - 요소를 자유롭게 배치하는 레이아웃

여러 요소를 순서와 순서와 상관없이 화면에 가로세로 좌표값을 기준으로 배치
특정 영역 안에서 위치 조절을 자유롭게 할 수 있어 유연한 레이아웃이 가능

  • relative : 적용된 요소의 현재 위치값을 기준으로 상대 위치를 지정한다.
    (기준값 - 부모요소에게 주되게 사용 또는 z-index 값을 적용할 시 사용)
  • absolute : 부모요소의 특정 구간을 기준으로 절대 위치를 지정한다.(움직이는 요소)
    부모요소의 relative를 지정하지 않을 경우 window를 기준으로 움직인다.
  • fixed : 웹브라우저를 기준으로 절대위치를 지정한다. (화면기준으로 보임)
