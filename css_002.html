박스속성

웹페이지 레이아웃시 가장 중요한속성

  • 요소크기설정 - width 와 height
  • 영역설정 - padding
  • 외곽선 설정 - border
  • 여백 설정 - margin

작성법

  • 전체적용 - 숫자와 단위를 함께 하나만 작성
  • 각자적용 - 숫자와 단위를 네번 써서 작성한다
    순서는 위,오른쪽,아래,왼쪽 상단에서 시계방향으로 돈다
    작성시, 쉼표를 쓰지않고 띄어쓰기를 작성한다
  • 세트적용 - 위아래/왼쪽 오른쪽 숫자와 단위를 2번써서 작성한다
    앞- 위아래, 뒤- 왼쪽오른쪽 각자적용과 마찬가지로 띄어 쓰기를 통해 작성한다
    • margin-top : 위쪽여백
    • margin-bottom : 아래쪽여백
    • margin-left : 왼쪽여백
    • margin-right : 오른쪽여백
    • padding-top : 위쪽영역
    • padding-bottom : 아래쪽영역
    • padding-left : 왼쪽영역
    • padding-right : 오른쪽영역

border설정방법

박스의 테두리를 말하며 padding 바깥쪽에 위치한다.
색상, 굵기, 모양으로 설정한다.

  • border-color : 색상기재
  • border-wihth : 숫자와 단위
  • border-stle
    • none - 선을없앰
    • solid - 실선
    • dotted - 동그라미점으로 점선만들기
    • dashed - 긴 사각형 점선
    • double - 실선두줄
    • groove - (액자형) - 테두리가 파인것 처럼 그려짐
    • ridge - (액자형) - 테두리가 튀어나온것처럼 그려짐 - groove 반대 스타일
    • inset - (액자형) - 내부요소가 파인것처럼그려짐
    • outset - (액자형) - 내부요소가 튀어나온것처럼 그려짐

작성법 - color,width,style을 한번에 쓸수도 있고 각자 쓸수도있다. 순번에 관계없음
border : red 2px solid ( 2px 너비의 빨간색 실선 )

코너곡선만들기 - border-radius

원을 만들거나 둥근 사각형을 제작할때 사용한다.
주의사항 - 짧은 변의 50%를 초과하지 않는다.

백그라운드 (배경설정)

배경색이나, 배경이미지를 삽입할때 사용한다.
많은 속성들을 한번에 기재할수도 있다.

  • background-color
    • 영문이름으로 작성 : red, black, white....
    • hex코드 작성법 : #fffff, #00000, #fff
      반복적으로 들어갈 경우 레드에서 하나 그린에서 하나 블루에서 하나씩 작성가능
    • rgb 수치값작성 : rgb(0,20,50)
    • rgb와 투명도를 같이 작성 : rgba(0,200,50,0.5)
    • backgroungd-repeat : 배경이미지 또는 컬러를 반복한다.
      • repeat - 가로, 세로형으로 모두 패턴처럼 깔림(타일)
      • repeat-x - 가로방향으로만 깔림 (벽지)
      • repeat-y - 세로방향으로만 깔림
      • no-repeat - 반복하지않음
  • background-image : url(이미지경로작성)
  • background-size : 배경이미지의 크기를 태그사이즈대비 변경하고자 할때 사용한다
    • contain - 이미지의 비율을 유지하면서 영역을 꽉 채움
    • cover - 배경이미지가 잘리더라도 화면의 영역을 꽉 채움
  • backgroungd-position : top, center, left 등 위치를 지칭하는 단어로도 사용가능하고 숫자와 단위로도 이동가능하다.
    왼쪽상단에서부터 거리가멀어지고 앞수 오른쪽으로 이동 다음수가 아랫쪽으로 이동 (20px 60px)
  • backgroungd-attachment : fixed 로 바닥에 붙이는 효과

그라디언트를 활용하여 배경색 또는 패턴을 제작할 수 있다.

  • linear-gradient-(각도,시작색,마지막색)
  • radial-gradient-(시작색, 마지막색)
  • patten - 직선그라데이션의 수직시작점을 퍼센트% 값으로 잡아 옆에 여백을 만들어 반복시켜적용
  • 그라이디언트헌트 바로가기
